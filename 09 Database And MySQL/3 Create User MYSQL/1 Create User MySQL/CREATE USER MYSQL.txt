CREATE USER MYSQL

1. GO TO MYSQL
   $mysql -u root -p

2. CREATE USER
   mysql> create user 'newuser'@'localhost' identified by '12345678';
   
   - If Condition ERROR :
     ERROR 1819 (HY000): Your password does not satisfy the current policy requirements

   Read Noted : 
   There are three levels of password validation policy enforced when Validate Password plugin is
   enabled:
   - LOW Length >= 8 characters.
   - MEDIUM Length >= 8, numeric, mixed case, and special characters.
   - STRONG Length >= 8, numeric, mixed case, special characters and dictionary file.

3. Check Level Password
   mysql> SHOW VARIABLES LIKE 'validate_password%';
   Default Level ==> Medium
   Solution ==> Create another password with combination (Level - MEDIUM Length >= 8, numeric, mixed case, and special characters.)

4. SHOW MYSQL USER
   mysql>select user from mysql.user;
   
5. Change Level Password To Create Simple Password
   mysql>set global validate_password_policy=0;
   or,
   mysql>set global validate_password_policy=LOW;
